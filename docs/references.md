## 4. Gợi ý cách trích dẫn trong báo cáo

Trong phần nội dung báo cáo, có thể trích dẫn theo số thứ tự tài liệu, ví dụ:

- Backend của hệ thống được xây dựng dựa trên Spring Boot và Spring Security để triển khai REST API, xác thực và phân quyền người dùng [1], [2].
- Các lớp entity và repository sử dụng Spring Data JPA, Hibernate ORM và MySQL để ánh xạ đối tượng - quan hệ và lưu trữ dữ liệu [3], [4], [5].
- Cơ chế xác thực sử dụng JSON Web Token theo chuẩn RFC 7519 và thư viện JJWT [6], [7].
- Giao diện người dùng được xây dựng bằng React, Vite và Tailwind CSS [10], [12], [13].
- Các biểu đồ trong báo cáo được mô tả theo cú pháp Mermaid, đặc biệt là biểu đồ lớp [18], [19].
